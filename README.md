# Прогнозирование рецидива диабетического кетоацидоза

Научно исследовательский проект (НИР), Университет ИТМО.

## Цель работы
Разработка интерпретируемой модели машинного обучения для прогнозирования рецидива диабетического кетоацидоза (ДКА) на основе клинических данных российской популяции.

## Используемые методы и инструменты
- Python
- Google Colab
- Pandas, NumPy
- Scikit-learn
- CatBoost, XGBoost, Random Forest, LogRegression
- SHAP
- Matplotlib, Seaborn

## Содержание репозитория
- `notebooks/` — Jupyter-ноутбуки с анализом данных и обучением моделей
- `presentation/` — презентация проекта
- `supporting docs/` — вспомогательные документы
